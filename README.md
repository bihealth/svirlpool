## Developer Notes

### Prerequisites

Install [`uv`](https://docs.astral.sh/uv/getting-started/installation/):

```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Instructions

Clone repository:

```
git clone git@github.com:bihealth/svirlpool.git
cd svirlpool
```

Run code formatting

```
make fix
```

Run formatting, lints, other static checks:

```
make check
```

Run tests:

```
make check
```

Or all in one:

```
make fix check test
```
